# My personal folio

A single-page site with projects, creative work and contact links.

**Live site:** [https://ndrdlc.github.io](https://ndrdlc.github.io)

## Contents

- **Personal projects** — links to GitHub repos and external sites
- **Interpolations** — short ML interpolation studies (video)
- **Videos** — additional video work
- **Photographs** — film photography grid
- **Stack** — languages, tools, and platforms
- **Contact** — email and social links

## Project structure

```
.
├── index.html          # Page markup, styles, and scripts (all-in-one)
└── assets/
    ├── photos/         # Portrait and photograph images
    ├── interpolations/ # Interpolation study videos
    └── videos/         # Additional video work
```

## Tech

- Plain HTML, CSS, and JavaScript (no framework)
- [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono) via Google Fonts
- Scroll-triggered reveals, a live clock, and viewport-aware video playback
- Respects `prefers-reduced-motion` for animations and autoplay
